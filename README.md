# Chrome Dinosaur Game (Java)

A simple Java Swing version of the Chrome Dinosaur game.  
The dino can jump, avoid cacti and birds, and the score increases over time.  
Includes clouds, moving ground, and improved collision detection.


## Features
- Dino jump with gravity effect  
- Cactus and bird obstacles  
- Clouds in the background  
- Score increases gradually  
- Game speed increases over time  
- Clean collision boxes (more accurate hits)
